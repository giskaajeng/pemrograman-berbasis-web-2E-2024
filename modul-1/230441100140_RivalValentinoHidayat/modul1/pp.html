# __Class, Object, Atribute, dan Method__
__Pengertian Class, Object, Atribute, dan Method__
*Class ialah template yang yang menggambarkan object dari class itu sendiri,
class juga termasuk dalam grup object dengan kesamaan atribut, brhaviour dan
relasi ke object lain.
*Object adalah instansiasi dari class. Jika secara umum class menampilkan
sebuah object, sebuah instansiasi ialah repsentasi nyata dari class tersebut.

_class_ tersebut.
*Atribute sendiri adalah data atau informasi yang disimpan ke dalam sebuah
_class_
